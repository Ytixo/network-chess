# Jeu d'échec en réseaux
Jeu d'échec en réseaux fait en python à l'école Polytech.
Pour y jouer lancer le serveur et 2 clients avec les commandes :

```
python3 server.py
```

```
python3 client.py (Si besoin changez l'ip dans Models/network.py)
```
